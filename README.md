Personal Introduction Program (Python Beginner Project).

 This is a beginner-friendly Python project that demonstrates basic programming concepts such as variables, user input, strings, lists, and formatted output.

Theory Concepts Covered

- What is Programming?
- Python as a Beginner-Friendly Language.
- Variables (Numbers, Strings, Boolean).
- Basic Math Operations (+, -, *, /).
- Strings and Text Manipulation.
- Lists and Accessing Items.
- Understanding Code Execution Step-by-Step.

Hands-On Practice

- Install Python.
- Write "Hello World".
- Create Variables.
- Perform Basic Math.
- Work with Lists.
- Build a Simple Bio-Data Program.

Project Overview
 
This program:
- Asks for user's name.
- Asks for user's age.
- Asks for user's favorite hobby.
- Displays a friendly welcome message.

Technical Requirements

- Uses input() for user interaction.
- Stores data in variables.
- Uses print() for output.
- Uses f-strings for formatting.
- Contains at least 3 questions.


How to Run the Program

1. Install Python (3.x recommended).
2. Run the code.
